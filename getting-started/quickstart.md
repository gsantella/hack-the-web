## Basics

In this course, you will learn the fundamentals of hacking the web. You will learn how to build with the building blocks of web design and learn how websites are crafted with _**code**_.

Students will explore the essential components of creating modern websites from scratch. The course covers the fundamentals of **HTML** for structuring web content, **CSS** for styling and layout, and **JavaScript** to add interactivity and dynamic elements.&#x20;

Additionally, students will learn about responsive design principles to ensure websites look great on all devices, as well as key accessibility and SEO best practices to enhance both usability and visibility on search engines. By the end of the course, learners will have the skills to build professional, user-friendly websites while utilizing the latest trends and tools in web development.
